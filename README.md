# My Main Site

This project retains much of the standard, scaffolded structure that `npx create-nuxt-app` provides.

## Requirements

* Node.js

## Getting Started

1. Clone the repo
2. `cd` into `mainsite`
3. Run `npm i`
4. Run `npm run dev`

## Notes

* Vuetify.js was used for Material Design pre-styled Vue components.
